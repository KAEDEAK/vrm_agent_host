General VRM Agent Host

Load and display VRM(VRoid)

Load VRM file (SET)

/?target=vrm&cmd=load&file=sample01.vrm

Camera Control

Toggle Orthographic Mode (SET)

/?target=camera&cmd=orthographic&enable=true

Set Orthographic Size (SET)

/?target=camera&cmd=orthographic&size=0.4

Camera auto adjust (adjust) (SET)

/?target=camera&cmd=adjust

Eye Control

Toggle LookAtHead (SET)

/?target=vrm&cmd=lookAtHead&enable=true

Audio Reactive Lip Sync (Volume-Based Animation)

Get Lip Sync status (GET)

/?target=lipSync&cmd=getstatus

Audio Reactive Lip Sync ON (SET)

/?target=lipSync&cmd=audiosync_on&channel=2

Audio Reactive Lip Sync OFF (SET)

/?target=lipSync&cmd=audiosync_off

VRMAnimation: Load and play

Play VRMA file (SET)

/?target=animation&cmd=play&file=VRMA_01.vrma&continue=false

Animation using Anime Girl Idle Animation

Get animation list/status (GET)

/?target=animation&cmd=getstatus
/?target=animation&cmd=getstatus&opt=alias

Play animation (play) (SET)

/?target=animation&cmd=play&id=Idle_Generic_01

Stop / Resume (SET)

/?target=animation&cmd=stop
/?target=animation&cmd=resume

VRoid Standard Blend Shapes

Apply Expression (shape) (SET)

/?target=animation&cmd=shape&word=Happy&seamless=y
/?target=animation&cmd=shape&word=reset&seamless=y

Mouth shape (shape) (SET)

Audio Reactive Lip Sync uses "Aa", so if it's active, "Aa" might not be applied. Please disable it first if needed.
/?target=animation&cmd=shape&word=Aa&seamless=y

Mouth shape (mouth aliases) (SET)

These aliases correspond to Aa, Ih, Ou, Ee, Oh.
/?target=animation&cmd=mouth&word=A

Blink Control

autoBlink (SET)

/?target=animation&cmd=autoBlink&enabled=true&freq=1500
/?target=animation&cmd=autoBlink&enabled=false

Manually set/reset Blink (SET)

/?target=animation&cmd=shape&blink=0.9,0.0
/?target=animation&cmd=shape&blink=reset
/?target=animation&cmd=reset_blink

Avatar Position & Rotation Commands (Test)

Get avatar position (getLoc) (GET)

/?target=vrm&cmd=getLoc

Set avatar position (setLoc) (SET)

/?target=vrm&cmd=setLoc&xyz=0,0.5,-1

Get avatar rotation (getRot) (GET)

/?target=vrm&cmd=getRot

Set avatar rotation (setRot) (SET)

/?target=vrm&cmd=setRot&xyz=0,45,0

Continuous Move / Rotate Commands (Test)

Continuous move (move) - with 'from' (SET)

The "duration" unit is msec (1000 = 1 second). "delta" splits movement into segments to define speed distribution.
/?target=vrm&cmd=move&from=-1,0,0&to=1,0,0&duration=1000&delta=20,60,20

Continuous move (move) - no 'from' (SET)

/?target=vrm&cmd=move&to=0,0,0&duration=1000&delta=20,60,20

Continuous rotate (rotate) - with 'from' (SET)

/?target=vrm&cmd=rotate&from=0,-45,0&to=0,45,0&duration=1000&delta=20,60,20

Continuous rotate (rotate) - no 'from' (SET)

/?target=vrm&cmd=rotate&to=0,0,0&duration=1000&delta=20,60,20

Stop move (stop_move) (SET)

/?target=vrm&cmd=stop_move

Stop rotate (stop_rotate) (SET)

/?target=vrm&cmd=stop_rotate

Background Image

Switch background image (SET)

/?target=background&cmd=load&file=sample01.png

Background color fill (test)

Change background color (fill color)

/?target=background&cmd=fill&color=0000FE
/?target=background&cmd=fill&color=FF00FF
/?target=background&cmd=fill&color=00FF00
/?target=background&cmd=fill&color=000000
/?target=background&cmd=fill&color=FFFFFF

Server commands

Make window transparent (transparent) (SET)

Enable transparency (with color):

/?target=server&cmd=transparent&color=0000FF&enable=true
/?target=server&cmd=transparent&color=FF00FF&enable=true
/?target=server&cmd=transparent&color=00FF00&enable=true
/?target=server&cmd=transparent&color=000000&enable=true
/?target=server&cmd=transparent&color=FFFFFF&enable=true

Disable transparency:

/?target=server&cmd=transparent&enable=false

Keep window always on top (stayOnTop) (SET)

/?target=server&cmd=stayOnTop&enable=true
/?target=server&cmd=stayOnTop&enable=false

Allow object drag (allowDragObjects) (SET)

/?target=server&cmd=allowDragObjects&enable=true
/?target=server&cmd=allowDragObjects&enable=false

Get server status (getstatus) (GET)

/?target=server&cmd=getstatus

Terminate server (terminate) (SET)

/?target=server&cmd=terminate
